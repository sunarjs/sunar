`accepts` in Sunar offer a fast and straightforward method to add basic restrictions to commands. They function like [protectors](/docs/builders/protector) but with simpler syntax, making them ideal for enforcing permissions and other basic requirements without needing to write complete middleware functions.

## Usage

```js
import { Slash, accepts, execute } from 'sunar';

const slash = new Slash({
	name: 'test',
	description: 'This a testing',
});

accepts(slash, { // [!code highlight]
	clientPermissions: ['SendMessages'], // [!code highlight]
}); // [!code highlight]

execute(slash, async (interaction) => {
	// handle execution
});

export { slash };
```
